# 공부시간 기록 웹앱 v3

포함 기능
- 날짜별 공부시간 입력
- 빠른 입력 버튼
- 오늘 / 이번 주 / 이번 달 / 연속 기록 요약
- 캘린더 히트맵 시각화
- 최근 30일 그래프
- JSON 백업 / 복원
- PWA 적용

GitHub 업로드 방법
1. 새 저장소를 만든다.
2. 이 폴더의 모든 파일을 저장소 루트에 업로드한다.
3. Settings → Pages → Deploy from a branch
4. main / root 선택 후 저장
5. 배포 주소를 사파리에서 열고 홈 화면에 추가

참고
- 데이터는 로컬 저장소에 저장된다.
- 백업 파일을 내려받아 두면 기기 변경 시 복원 가능하다.
