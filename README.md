# 공부시간 기록 웹앱

아이폰 사파리와 GitHub Pages에서 바로 사용할 수 있는 심플한 공부시간 기록 웹앱입니다.

## 포함 파일
- `index.html` : 앱 본체
- `.nojekyll` : GitHub Pages 배포 보조 파일

## GitHub 업로드 방법
1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일들을 저장소 루트에 업로드합니다.
3. GitHub 저장소의 **Settings → Pages**로 이동합니다.
4. **Deploy from a branch**를 선택합니다.
5. 브랜치는 `main`, 폴더는 `/ (root)`로 선택하고 저장합니다.
6. 잠시 후 배포 주소가 생성됩니다.

## 아이폰에서 앱처럼 쓰는 방법
1. 배포된 GitHub Pages 주소를 사파리에서 엽니다.
2. 하단 공유 버튼을 누릅니다.
3. **홈 화면에 추가**를 선택합니다.

## 현재 기능
- 날짜별 공부시간 입력
- 메모 기록
- 캘린더 보기
- 시험일/일정 입력
- 최근 30일 그래프
- 브라우저 로컬 저장

## 참고
기록 데이터는 브라우저의 로컬 저장소에 저장됩니다.
따라서 같은 주소라도 브라우저 데이터를 삭제하면 기록이 사라질 수 있습니다.
