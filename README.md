# 공부시간 기록 웹앱

## 포함 파일
- `index.html`
- `README.md`
- `.nojekyll`

## GitHub 업로드 방법
1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일들을 저장소 루트에 업로드합니다.
3. Settings → Pages로 이동합니다.
4. Deploy from a branch를 선택합니다.
5. main / root를 선택하고 저장합니다.

## 다음 버전에서 추천하는 개선점
- 월간 캘린더에서 공부시간이 많은 날은 색을 진하게 표시
- 주간/월간 합계 보기 추가
- 공부시간 빠른 입력 버튼 추가
- 데이터 백업/복원 기능 추가
- 아이폰 홈화면 앱 아이콘 및 설치형(PWA) 적용
